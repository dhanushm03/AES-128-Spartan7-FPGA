
# 🚀 AES-128 Spartan-7 FPGA Implementation

This repository contains a complete RTL implementation of the AES-128 encryption algorithm using Verilog HDL and deployed on a Spartan-7 FPGA.

The project demonstrates:

- Modular RTL design
- Hardware synthesis & implementation
- FPGA programming
- Physical output verification using on-board display

---

## 📂 Repository Structure

```
AES-128-Spartan7-FPGA/
│
├── AES-128/        → Complete project files
├── LICENSE         → Apache-2.0 License
└── README.md       → Project overview
```

---

## 📌 Project Highlights

- AES-128 encryption implemented in Verilog
- Modular architecture (SubBytes, ShiftRows, MixColumns, Key Expansion)
- Spartan-7 FPGA deployment
- Seven-segment output verification
- Hardware demonstration video included

---

## 🛠 Tools Used

- Vivado Design Suite
- Verilog HDL
- Spartan-7 FPGA Board

---

## ▶️ How to Use

1. Navigate to the `AES-128` folder.
2. Follow the detailed implementation steps in that folder’s README.
3. Add source files and constraints in Vivado.
4. Generate bitstream and program the FPGA.
5. Verify output using the provided hardware demo reference.
