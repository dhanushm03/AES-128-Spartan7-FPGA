
# 🎥 Hardware Output Demonstration

## 📌 Demonstration Includes

- FPGA board powered ON
- Reset initialization
- Start signal activation
- AES encryption execution
- Done LED indication
- Final encrypted output on seven-segment display

---

## ✅ Verification Method

Correct functionality confirmed by:

1. Successful encryption completion.
2. Done signal activation.
3. Observed encrypted output on hardware display.

This validates successful synthesis, implementation, and FPGA programming.
